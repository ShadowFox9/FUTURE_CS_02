# FUTURE_CS_02
## Web Application Vulnerability Assessment

This project documents a hands-on vulnerability assessment of a deliberately vulnerable web application, conducted in alignment with the OWASP Top 10 security risks.

### Objectives
- Identify common web application vulnerabilities
- Map findings to OWASP Top 10
- Provide remediation recommendations
- Produce a professional security assessment report

### Tools
- OWASP Juice Shop
- OWASP ZAP
- Burp Suite (Community)
